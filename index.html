How It Works

자료를 받아
소개 구조부터 정리합니다.

빠르게 만드는 것보다, 먼저 무엇을 어떤 순서로 보여줄지 설계합니다.

01

방향 상담

업종, 역할, 소개 목적, 공유 대상자를 확인합니다. 미팅 전 자료인지, 대표 소개인지, 사업 소개인지 먼저 정리합니다.

접수 후 1영업일 이내 회신
02

자료 정리

프로필 사진, 이력, 서비스 설명, 포트폴리오, 기존 소개서 등을 받습니다. 부족한 정보는 질문 리스트로 보완합니다.

소요시간: 1~2일
03

구조 설계

첫 화면 메시지, 신뢰 근거, 서비스 설명, 사례, 문의 동선을 어떤 순서로 보여줄지 구성합니다.

핵심: 카피 구조와 섹션 흐름 PRO 이상: 확장 섹션 설계
04

시안 제작

정리된 구조를 바탕으로 반응형 웹페이지 시안을 제작합니다. 실제 URL 미리보기로 PC와 모바일 화면을 확인합니다.

소요시간: 3~5일
05

수정 반영

문구, 순서, 이미지, 버튼 연결을 함께 점검합니다. 수정사항은 섹션별로 정리해 반영합니다.

수정 범위는 플랜별 상이
06

납품과 연결

완성 URL과 소스, 필요 시 QR 연결 이미지를 전달합니다. 명함, 제안서, 카카오톡 프로필, 이메일 서명에 활용할 수 있게 안내합니다.

도메인 연결은 별도 상담

자료 확정 후 보통 1~2주 내 진행
