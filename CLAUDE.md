# WeThru 비즈니스 프로필 랜딩페이지 프로젝트

## 프로젝트 목적
크몽·숨고 등 외주 플랫폼에서 "비즈니스 프로필 + 명함" 상품(BASIC 20만원~)을 판매하기 위한
외부 HTML 랜딩페이지(B형 페이지). 네이버·메타 광고 도착 URL로도 사용.

## 타깃 고객
- 스타트업 대표, 프리랜서, 1인 기업가, 임원
- 25~45세, 개인 브랜딩·온라인 명함에 관심
- "검색당하는 시대"에 자신을 전문적으로 보여주고 싶은 사람

## 비즈니스 맥락
- 이 페이지는 "쌍둥이 퍼널" 중 B형(프로필 전용)
- 상단 70%: 비즈니스 프로필·명함 집중
- 하단 30%: 사업계획서·IR·PPT·홈페이지 등 위쓰루 나머지 서비스 교차 업셀링
- A형 페이지(사업계획서 메인)는 별도 제작 예정

## 기술 스택
- 순수 HTML + CSS + Vanilla JS (프레임워크 없음)
- GitHub Pages 배포 (정적 파일)
- 반응형 (375px ~ 1440px)
- 폰트: Pretendard(본문), 영문 디스플레이는 자유 선택 (단, Inter·Roboto·Arial 금지)
- 애니메이션: CSS 위주, 필요 시 GSAP CDN 또는 IntersectionObserver
- 성능: Lighthouse 90+, FCP ≤ 1.5s

## 절대 금지
- Inter, Roboto, Arial, Space Grotesk 폰트
- 보라색 그라디언트 on 화이트 배경
- 뻔한 카드 그리드 레이아웃
- "AI가 만든 느낌"의 일반적 SaaS 템플릿 디자인
- 외부 연락처(전화번호, 이메일) 직접 노출 (크몽 정책)
